# 📊 Análise de Vendas por Região e Produto

Projeto de análise exploratória identificando padrões de lucro
e ROI em dados de vendas regionais.

## Ferramentas
Python | pandas | matplotlib | seaborn

## O que foi analisado
- Sazonalidade mensal de vendas
- Lucro e ROI por produto
- Desempenho por micro região
- Cruzamento produto x região (heatmap)
